# Проект: Практика Java — Методы, Условия, Циклы, Массивы

## Описание проекта
Данный проект представляет собой учебную работу на Java для закрепления базовых тем:
- Методы
- Условия (`if`, `switch`)
- Циклы (`for`, `while`)
- Массивы

Программа состоит из двух файлов:
- `Methods.java` — содержит 20 методов для разных задач, разделённых на 4 категории (Методы, Условия, Циклы, Массивы)
- `Main.java` — главное меню для выбора раздела и задачи, обработка пользовательского ввода

Программа позволяет пользователю выбирать раздел и задачу, вводить данные, после чего выводит результат. Ввод проверяется и защищён от некорректных данных (числа вне диапазона, пустые строки, неверные символы).

## Требования
- Java JDK 8 или выше
- Любая IDE (IntelliJ IDEA, Eclipse) или терминал
- Отсутствие внешних библиотек — используются только стандартные классы Java
- Код оформлен по стандарту Google Java Style и принципам OOP (DRY, KISS, SoC, SOLID)

## Алгоритм и идея решения
1. Класс `Methods` содержит методы для каждой задачи. Методы реализованы так, чтобы использовать только конструкции, соответствующие разделу:
   - Методы — обычные методы класса, работа с числами и символами
   - Условия — `if`, `switch` без использования Math.max/Math.min
   - Циклы — `for` и `while`
   - Массивы — ручная работа с массивами (поиск, вставка, реверс, индексация)
2. Класс `Main` реализует многоуровневое меню: выбор раздела, затем задачи
3. Пользователь вводит данные через консоль, программа проверяет корректность и выводит результат
4. Для задач с графическим выводом (квадрат, треугольник) используется `System.out.println()`

## Инструкции

2. Запустите программу:

Следуйте инструкциям на экране:

## Задачи
Раздел 1: Методы


### 1. Дробная часть числа 

Сигнатура метода: public double fraction(double x);

Описание: Возвращает дробную часть числа x (x - (int)x).

Пример 1: x = 6.78 → результат: 0.78

Пример 2: x = -3.5 → результат: 0.5


### 2. Букву в число

Сигнатура метода: public int charToNum(char x);

Описание: Преобразует символ-цифру в число. Если символ не цифра — возвращает -1.

Пример: '7' → 7, 'a' → -1


### 3. Двузначное число

Сигнатура метода: public boolean is2Digits(int x);

Описание: Проверяет, является ли число двузначным.

Пример: 15 → true, 7 → false


### 4. Диапазон

Сигнатура метода: public boolean isInRange(int a, int b, int num);

Описание: Проверяет, входит ли num в диапазон [a, b].

Пример: a = 5, b = 10, num = 7 → true


### 5. Равенство трёх чисел

Сигнатура метода: public boolean isEqual(int a, int b, int c);

Описание: Проверяет, равны ли все три числа.

Пример: a = 3, b = 3, c = 3 → true, a = 3, b = 4, c = 3 → false


## Раздел 2: Условия


### 6. Модуль числа

Сигнатура метода: public int abs(int x);

Описание: Возвращает модуль числа x.

Пример: x = 5 → 5, x = -3 → 3


## 7. Делимость на 3 или 5

Сигнатура метода: public boolean is35(int x);

Описание: Проверяет, делится ли число на 3 или 5, но не на оба одновременно.

Пример: 9 → true, 10 → true, 15 → false


## 8. Тройной максимум

Сигнатура метода: public int max3(int x, int y, int z);

Описание: Находит максимальное из трёх чисел, используя только условия if.

Пример: 5, 7, 3 → 7


## 9. Двойная сумма

Сигнатура метода: public int sum2(int x, int y);

Описание: Возвращает сумму x и y. Если сумма в диапазоне 10–19, возвращает 20.

Пример: 7 + 5 → 20, 3 + 4 → 7


## 10. День недели (через switch)

Сигнатура метода: public String day(int x);

Описание: Возвращает название дня недели по номеру от 1 до 7.

Пример: 1 → "понедельник", 7 → "воскресенье", 9 → "это не день недели"



## Раздел 3: Циклы


## 11. Числа подряд

Сигнатура метода: public String listNums(int x);

Описание: Возвращает строку с числами от 0 до x включительно, через пробел.

Пример: x = 5 → "0 1 2 3 4 5"


## 12. Чётные числа

Сигнатура метода: public String chet(int x);

Описание: Возвращает строку с чётными числами от 0 до x включительно.

Пример: x = 6 → "0 2 4 6"


## 13. Длина числа

Сигнатура метода: public int numLen(long x);

Описание: Возвращает количество цифр в числе.

Пример: 12345 → 5, -987 → 3


## 14. Квадрат из символов

Сигнатура метода: public void square(int x);

Описание: Выводит квадрат из символов * размера x на x.


## 15. Правый треугольник
Сигнатура метода: public void rightTriangle(int x);

Описание: Выводит правый треугольник из * высоты x.


## Раздел 4: Массивы

## 16. Поиск первого значения

Сигнатура метода: public int findFirst(int[] arr, int x);

Описание: Возвращает индекс первого вхождения x в массиве arr. Если нет — -1.

Пример: arr = {1,2,3,2}, x = 2 → 1


## 17. Максимум по модулю

Сигнатура метода: public int maxAbs(int[] arr);

Описание: Возвращает элемент массива с максимальным модулем.

Пример: {1, -4, 3} → -4


## 18. Добавление массива в массив

Сигнатура метода: public int[] add(int[] arr, int[] ins, int pos);

Описание: Вставляет массив ins в массив arr на позицию pos.

Пример: arr = {1,2,3}, ins = {7,8}, pos = 1 → {1,7,8,2,3}


## 19. Реверс массива

Сигнатура метода: public int[] reverseBack(int[] arr);

Описание: Возвращает новый массив, обратный arr.

Пример: {1,2,3} → {3,2,1}


## 20. Все вхождения числа

Сигнатура метода: public int[] findAll(int[] arr, int x);

Описание: Возвращает массив индексов, где встречается число x в массиве arr.

Пример: arr = {1,2,1,3}, x = 1 → {0,2}
